
Usage: run-book ontology reference [options] [command]

Reference data such as 'the git repo for this service is here'

Options:
  -h, --help      display help for command

Commands:
  all             Reference data such as 'the git repo for this service is
                  here'
  for <thing>     Retreive the reference data about one thing, For example
                  'leo:service' means get me all the reference data about the
                  service leo
  help [command]  display help for command
