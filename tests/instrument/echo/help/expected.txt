Usage: run-book instrument echo [options]

An instrument that sends its arguments to standard output

Options:
  -p|--params <params...>  a space separated name1:value1 name2:value2
  -s|--showCmd             Show the command instead of executing it
  -r|--raw                 Show the raw output instead of formatting it
  -j|--json                Show the output as json
  --onelinejson            Show the output as json
  -1|--oneperlinejson      Show the output as json
  --debug                  include debug info
  --config                 Show the json representing the command in the config
  -h, --help               display help for command
