
Usage: run-book view [options] [command]

Commands to work with views which allow you to find things out about systems

Options:
  -h, --help            display help for command

Commands:
  displayGit [options]  Lists the git properties of the services
  domainCat [options]   Display a csv file for each dmain in each environment
  domain [options]      Lists the domain and port of the services
  help [command]        display help for command
